You can easily integrate with Esendex's Echo application using our jQuery plugin. Here's how:

  • Add jQuery to you page (jquery-1.8 or later is supported)
  • Include the jquery.esendex.sms.js file
  • Apply the esendexSms function to an element or class
  • Include a phone number, default message or cssClass as part of the markup

Any html element (e.g. a <span> element) which contains just a phone number can have the plugin applied directly e.g. +44 7800 000 000

You can specify a recipient using the 'data-esendex-phone-number' attribute, allowing you to customize the link text displayed to the user e.g. Send SMS message

You can add a message as part of your HTML markup using the 'data-esendex-message' attribute e.g. Send Default Message

If you have multiple accounts then you can pick which account to send from using the 'data-esendex-account' attribute. Send Default Message From Specific Account
